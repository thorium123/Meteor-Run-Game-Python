# Meteor-Run-Game-Python
Meteor Run is a python game that can run on most operating systems using the pygame package. 

## Installation
To install Meteor Run use the `git clone https://github.com/thorium123/Meteor-Run-Game-Python.git` to curl the files.

To run Meteor Run the following dependancies are needed to be installed. Click [here](https://github.com/thorium123/Meteor-Run-Game-Python/blob/main/Dependancies.txt) to see the list of dependancies.

## Game Play
Meteor Run is a relatively easy game to play. It is played using WSAD or the arrow keys to move your spaceship left to right. The aim of the game is to last as long as you can whilst not getting hit by a meteor.
